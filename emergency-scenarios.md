# הפיכת האפליקציה לכלי חירום

## רשת Mesh מקומית ללא תשתית חיצונית
- **APK ייעודי** מפעיל נקודת־חמה עם SSID וסיסמה קבועים.
- כרטיס הרשת נדלק אוטומטית ומכשיר מחפש רשתות זהות של משתמשים אחרים.
- התוצאה: רשת מקומית עצמאית ("שכונה" מחוברת) גם כשאין חשמל או ממסרי תקשורת.
- כל מכשיר מריץ את היישום ומשמש כראוטר/ריפיטר לניתוב המסרים הלאה.

## שימוש בחיישני המכשיר למצבי חירום
1. **ג'ירוסקופ / מד תאוצה:**
   - גילוי נפילות או תנועה חריגה ושיגור התראה לשאר המשתמשים.
   - מצפן/כוון-דרך בלי GPS: בניית ניווט בסיסי על סמך כיוון המכשיר.
2. **מגנטומטר:**
   - בדיקת שדה מגנטי חריג (דיווח על מתכות/כבלי חשמל תת-קרקעיים).
   - מאפשר קביעת כיוון כשאין שירותי מיקום.
3. **מד תאוצה + מיקרופון:**
   - זיהוי טפיחות/רעידות מאחורי קיר (אינדיקציה לחיפוש ניצולים).
   - איסוף נתוני זעזועים מאסיבי לזיהוי רעידות אדמה מקומיות.
4. **מצלמה + חיישן זווית + GPS (אם קיים):**
   - "סורק נזק" – צילום מצורף נתוני כיוון וסטטוס לשליחת דיווח לרשת.
   - בדיקת שיפוע/סטייה של עמודים או קירות.
5. **פנס / רטט:**
   - מצב SOS אוטומטי (הבהוב קבוע או דפוס רטט).
   - חיווי רטט כאשר רשת ה-Mesh מזהה קרבה של משתמש אחר בצוות.

## שלבי מימוש באפליקציית APK
1. שימוש ב-`SensorManager` (Android) לגישה לחיישנים ולחיבורי רשת.
2. בקשת הרשאות רלוונטיות בזמן ריצה (מיקום, חיישנים, מצלמה וכו').
3. יצירת UI למצבי חירום: "סריקה", "בדיקת תנועה", "מצב SOS" וכו'.
4. שילוב עם שכבת הרשת הקיימת (Mesh) לשיתוף מדידות והתרעות בין המשתמשים.

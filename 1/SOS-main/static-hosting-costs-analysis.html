GitHub Pages

מפרט טכני:

  • ✓ חינם לחלוטין
  • ✓ עד 1GB אחסון
  • ✓ 100GB תעבורה חודשית (soft limit)
  • ✓ CDN גלובלי מובנה
  • ✓ SSL אוטומטי
  • ✓ Custom domains

עלות לפי מספר משתמשים:

10K משתמשים: חינם (30GB < 100GB)

50K משתמשים: חינם (150GB > 100GB - צריך התראה)

100K משתמשים: צריך פתרון תשלום

300K משתמשים: לא מתאים

מסקנה: מצוין עד 50K משתמשים, אחרי זה צריך לשקול אלטרנטיבות

  • ✓ CDN גלובלי
  • ✓ SSL אוטומטי
  • ✓ Custom domains

עלות לפי מספר משתמשים:

50K משתמשים: חינם (150GB > 100GB - צריך תשלום)

100K משתמשים: ~$20/חודש (תוספת תעבורה)

300K משתמשים: ~$80/חודש

מסקנה: מתאים עד 100K משתמשים, יקר מעבר לזה

המלצות למשקיעים

עד 50K משתמשים:

  • • GitHub Pages - הכי זול ומתאים להתחלה
  • • Netlify Free - חוויית מפתח טובה יותר

100K-300K משתמשים:

  • • AWS S3 + CloudFront - הכי כלכלי וגמיש
  • • Netlify Pro - פשוט לניהול אבל יקר יותר

לסיכום: ל-SOS מומלץ להתחיל עם GitHub Pages ולעבור ל-AWS כשהטראפיק גודל. העלות הכוללת תהיה פחות מ-$100 לחודש גם ב-300K משתמשים.

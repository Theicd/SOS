מה זה SPEAR ONE

טכנולוגיית ליבה מבוזרת שמוכיחה שאפשר להפעיל מוצרי אינטרנט שלמים – רשת חברתית, תקשורת, אחסון ו-AI – על שרת סטטי יחיד, בלי Back‑End ובלי DB. הסנכרון מתבצע דרך Relay Mesh (Nostr), המפתחות נשארים אצל המשתמש, והמערכת גדלה יחד עם הקהילה.

מה SPEAR ONE נותנת בשורה אחת

תשתית אינטרנט מבוזרת שמספקת מוצר מוכן לשימוש – ללא תלות בענן או שרת מרכזי.

  • קוד פרונט טהור שנפרס ל-Netlify / GitHub Pages / IPFS ומספק חוויית משתמש מלאה.
  • Relay Mesh (Nostr) לתמסורת וסנכרון – הנתונים נשמרים ומופצים ע"י המשתמשים והשותפים.
  • SphereOne Social – רשת חברתית אמיתית שמדגימה את הטכנולוגיה ואת המודל העסקי.

איך זה עובד בפועל

הפריסה כולה סטטית, הסנכרון מבוזר, והבקרה מתבצעת מקומית.

  1. Static Core נטען מהוסט סטטי ומגיש את כל ה-UI והלוגיקה.
  2. Relay Mesh מפיץ ומשמר את הפעילות החברתית ואת האחסון.
  3. Sentry AI פועל אצל המשתמש – סינון תכנים, אבטחה וקהילה נקייה.

מפת שכבות – מה מרכיב את SPEAR ONE

כל שכבה מוסיפה יכולת עסקית וטכנולוגית, ומהווה בסיס לשיתופי פעולה.

Static Core

Front-end מלא בקוד פתוח – אין Back‑End ואין צורך ב-DB.

Relay Layer

רשת Relays ציבורית/קהילתית שמספקת עמידות ותעבורה.

AI Layer

Sentry AI מקומי לניהול תכנים, מניעת ספאם והגנות פרטיות.

Apps

SphereOne Social + הרחבות (Chat, Call, Builder) – חוויות משתמש מוכנות.
